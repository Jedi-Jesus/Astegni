## KEY DIMENSIONS & SPECIFICATIONS

### Images
- **Cover Photo**: 1200x300px (Recommended)
- **Profile Avatar**: 400x400px (Recommended)
- **Maximum File Size**: 5MB

### Colors (From CSS Variables)
- Uses `var(--card-bg)`, `var(--text-primary)`, `var(--button-bg)`, etc.
- Theme-aware (light/dark mode support)
- Defined in `css/root/theme.css`

### Z-Index Layers
1. Base Content: z-index: 1
2. Sidebar: z-index: auto
3. Navigation: z-index: auto (fixed)
4. FAB: z-index: 1000
5. Rating Tooltip: z-index: 99999
6. Upload Modals: z-index: 10000

### Breakpoints (Tailwind)
- Mobile: < 768px
- Tablet: 768px - 1024px
- Desktop: > 1024px

### Animations
- **Hero Stats**: Counting animation (IntersectionObserver)
- **Study Tips Ticker**: 18s cubic-bezier scroll
- **FAB Items**: fadeInUp stagger (0.1s delay each)
- **News Carousel**: Auto-rotate or manual navigation
- **Footer Wave**: SVG path animation
