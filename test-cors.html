Instructions to Fix CORS Issues

  1. Make sure the backend is running on port 8000
  2. Clear your browser cache: Ctrl+Shift+Delete
  3. Hard refresh: Ctrl+Shift+R or Ctrl+F5
  4. Check browser console for errors: F12
