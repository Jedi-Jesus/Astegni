HOW IT WORKS (LAZY LOADING)
================================================================================

1. Page loads with modal-loader.js included
2. User clicks to open a modal (e.g., "Edit Profile")
3. Modal loader checks if modal is already loaded
4. If not loaded:
   - Fetches modal HTML from modals/tutor-profile/edit-profile-modal.html
   - Inserts into #modal-container
   - Caches for future use
5. Modal displays instantly

Result: Faster page load + modals load only when needed!
