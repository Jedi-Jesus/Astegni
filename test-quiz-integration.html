üß™ How to Test

  1. Open Quiz Modal: Click the "Open Quiz Manager" button above
  2. Create a Quiz: Click "Give a Quiz" button
  3. Search Student: Type in the student search field (e.g., "Abebe")
  4. Select Student: Click on a student from the dropdown
  5. Fill Details: Enter course name, quiz type, time, and days
  6. Add Questions: Click "Add Question" button
  7. Use Rich Editor: Type question text, use formatting buttons
  8. Select Question Type: Choose Multiple Choice, True/False, or Open Ended
  9. Fill Options: Add choices/answers based on question type
  10. Save or Post: Click "Save Quiz" (draft) or "Post Quiz" (publish)
  11. View in Sidebar: Your quiz appears in the left sidebar
  12. Close Modal: Press ESC or click overlay to close
